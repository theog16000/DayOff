# 🌴 DayOff - Gestionnaire de Congés pour WordPress

**DayOff** est un plugin WordPress léger et intuitif conçu pour simplifier la gestion des congés et des absences au sein d'une équipe. Il permet aux employés de soumettre des demandes et aux administrateurs de les valider via une interface moderne.

## ✨ Fonctionnalités

### 👤 Interface Collaborateur (Dashboard)
- **Tableau de Bord interactif** : Visualisation immédiate des soldes restants (CP, RTT, Récupération, Maladie).
- **Soumission de Demande** : Formulaire dynamique pour poser des congés sans recharger la page.
- **Calendrier Personnel** : Vue mensuelle intégrée pour suivre ses absences.
- **Paramètres** : Gestion autonome du profil (nom, email) et changement de mot de passe sécurisé.

### 👑 Interface Administrateur
- **Gestion des Demandes** : Interface dédiée pour approuver ou rejeter les demandes en un clic.
- **Suivi Global** : Vue d'ensemble des absences de l'équipe (en cours de développement).
